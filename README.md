# An AI-powered email intelligence tool designed to scan, categorize, and prioritize high-value opportunities directly from your inbox.

Built during a 2026 hackathon, this copilot uses LLMs to filter through the noise, extracting actionable leads and professional opportunities while automating routine responses. It transforms a cluttered inbox into a streamlined feed of what actually matters.

Key Features:

Automated Opportunity Detection: Identifies internships, job leads, and collaboration requests using agentic AI.

Context-Aware Categorization: Sorts incoming mail based on priority and relevance.

Seamless Integration: Designed to sit on top of your existing mail flow to provide real-time assistance.

Minimalist Architecture: Optimized for speed and clear, actionable insights.
